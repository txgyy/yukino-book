# NodeJS环境搭建

## nvm
版本切换工具
### 安装nvm
```shell
brew install nvm
```

### 查看nodejs远程列表
```shell
nvm ls-remote --lts
``` 
### 查看nodejs本地列表
```shell
nvm ls
```
### 切换版本
```shell
nvm use xxx
```

## nrm
镜像管理工具

### 安装nrm
```shell
brew install nrm
```

### 查看镜像列表
```shell
nrm ls
```

### 使用镜像
```shell
nrm use xxx
```
